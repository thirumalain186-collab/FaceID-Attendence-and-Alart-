# Smart Attendance System

## Installation

```bash
pip install opencv-python numpy Pillow pandas flask
```

## Quick Start

```bash
# 1. Register people
python register_faces.py

# 2. Train model
python train.py

# 3. Start attendance
python main.py
```

## Features
- Face Registration (LBPH)
- Real-time Recognition
- Email Alerts to Advisor & HOD
- Daily Reports
- SQLite Database
- Web Dashboard (Optional)
